<h1 align="center">
Vue Express Template
</h1>

<p align="center">
🚀 A monorepo with Vue 3.x frontend and Express 4.x backend.
<p>

<p align="center">
用于快速开启一个小型的前后端分离、共同管理的基础模板项目
<p>


## 📦 Installation

```
pnpm install
```

## 🚀 Running

```
pnpm start
```

🎡 Playground: http://localhost:4500/


## 🤟 Update Dependencies

```
$ pnpm --recursive update
$ pnpm --recursive update <package-name> --latest
```
